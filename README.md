# XCOM2 角色池工具 (XCOM2CharacterPoolUtility)

一个独立的工具，用于创建角色、合并池以及自定义XCOM 2角色池界面上可用的所有选项。

**原仓库**: [Shaosil/XCOM2CharacterPoolUtility](https://github.com/Shaosil/XCOM2CharacterPoolUtility)  
**原作者**: Shaosil

本版本在原项目基础上增加了中文语言支持，使用了ai辅助。

## 安装说明
* 请确保已安装 .NET Desktop Runtime 9.0。<a href="https://dotnet.microsoft.com/en-us/download/dotnet/thank-you/runtime-desktop-9.0.6-windows-x64-installer">点击此处从微软官网下载</a>
* 下载本程序的最新版本<a href="https://github.com/Shaosil/XCOM2CharacterPoolUtility/releases/latest">点击此处下载</a>，然后解压到任意文件夹
* 运行 XCOM 2 Character Pool Utility.exe

## 角色池保存位置

要让XCOM 2识别角色池，您需要将其保存到正确的位置。

如果您<b>没有</b>拥有"选中者的崛起"(War of the Chosen)扩展包，可以保存到
> {您的用户文件夹}\Documents\My Games\XCOM2\XComGame\CharacterPool\Importable

如果您<b>拥有</b>"选中者的崛起"(War of the Chosen)扩展包，除了上述路径外，您还可以保存到
> {您的用户文件夹}\Documents\My Games\XCOM2 War of the Chosen\XComGame\CharacterPool\Importable
